# EduPro Dashboard

A simple Streamlit app to analyze course enrollments, revenue, and user data.

# Features

* Upload your own Excel file (UserID, Age, Gender)
* Auto-generate sample data if no file is uploaded
* View total enrollments
* View revenue analysis
* View course performance
* View age group analysis
* Download generated data as CSV

---

# Technologies Used

* Python
* Streamlit
* Pandas
* NumPy

---

# Steps to Run Locally

Step 1: Install required libraries

```bash
pip install streamlit pandas numpy openpyxl
```

Step 2: Run the application

```bash
streamlit run streamlit_app.py
```

---

# Steps to Deploy

Step 1:

Upload project files to GitHub

Step 2:

Open Streamlit Cloud

Step 3:

Select your GitHub repository

Step 4:

Set main file path:

```bash
streamlit_app.py
```

Step 5:

Click Deploy

---

# Input File Format (Optional)

Excel file should contain:

* UserID
* Age
* Gender (optional)
